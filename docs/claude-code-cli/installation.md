## Installation

There are multiple ways to install the `claude` CLI:

::: code-group

```shell [Homebrew]
brew install claude
```

```shell [npm]
npm install -g @anthropic/claude-code
```

```shell [Shell]
curl -fsSL https://claude.ai/install.sh | bash
```

:::

## Verification

You can test your installation as follows:

```shell
claude --version
```
