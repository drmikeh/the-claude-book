# Welcome

Learning AI can be like drinking from 10 firehoses all at once. Not only is the AI universe vast, it keeps growing and changing.

This online book is intended to help you on your AI learning odyssey by:

1. Explaining concepts as briefly as possible
2. Using many analogies and visualizations
3. Navigating a learning journey that builds concepts in a logical andn intuitive way
4. Keeping things fun with hands-on examples
